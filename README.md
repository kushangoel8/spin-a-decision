# 🎡 Spin a Decision

A tiny, single-page decision wheel. Can't pick between options? Type them in, spin, and let chance decide.

## Try it

Open `index.html` in any browser — no build step, no dependencies. Pure HTML/CSS/JS.

Or enable **GitHub Pages** for this repo (Settings → Pages → Deploy from branch `main` / root) to get a live link.

## Features

- Customizable list of options (add/remove freely)
- Smooth animated spin with realistic easing
- Randomly weighted, unbiased selection
- Zero dependencies, zero build tools
